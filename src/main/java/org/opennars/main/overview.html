Package Specification

The code for distribution is NARS.jar, in Java 1.7.0_21.

It works both as :

  • an application (with full functionality) ;
  • an applet (with restricted functionality and display);
  • a command line batch tool

To run the application from the command line, go to the folder containing NARS.jar and type the following: 

            java -cp NARS.jar org.opennars.main_nogui.NARSBatch org.nars-dist/Examples/Example-NAL1-edited.txt
For more information, see NARS website, open-source project, discussion group
